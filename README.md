# Nexus CDK

> ⚠️ This project is experimental!
>
> It serves the purpose of experimenting with the DX for cloud applications, so I suggest not to use it.

## Requirements

- [Bun](https://bun.sh/)
- [pnpm](https://pnpm.io/)

## Development

```bash
bun dev
```

The command above will start the example project located in `example/`.
